# ⚽ JerseyShop - Fullstack Challenge E-commerce

Aplicación de e-commerce de jerseys de fútbol desarrollada con React + Vite.

## 🚀 Tecnologías
- React.js
- Vite
- Tailwind CSS
- Zustand
- Axios
- React Router DOM

## 📦 Instalación

```bash
npm install
npm run dev
```

## 🛍 Funcionalidades
- Galería de jerseys por posición
- Búsqueda en tiempo real
- Filtrado por categoría
- Paginación de productos
- Carrito de compras
- Checkout con resumen
- Login y Registro de usuarios
- Persistencia con localStorage
